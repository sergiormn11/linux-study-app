# Linux Study App

Página web diseñada para organizar, estudiar y practicar apuntes de un curso de Linux de forma clara, estructurada y progresiva.

Este proyecto funciona como una pequeña plataforma personal de estudio donde se pueden guardar apuntes, consultar comandos importantes, repasar teoría y practicar con ejercicios relacionados con la terminal de Linux.
